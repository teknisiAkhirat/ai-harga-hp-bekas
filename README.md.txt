# Estimasi Harga HP Bekas dengan AI

Website sederhana untuk memprediksi harga HP bekas berdasarkan kondisi fisik dan tahun rilis. Dibuat dengan Python + Streamlit.

## Input Form:
- Merk dan tipe HP
- Tahun rilis
- Apakah LCD retak
- Apakah baterai soak
- Apakah ada dus

## Teknologi:
- Python
- Streamlit
- Scikit-learn (Linear Regression)
